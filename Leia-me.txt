O jogo LagganDino foi criado para rodar especificamente
no sistema operacional Linux Ubuntu 18.04 ou maior.

Para rodar o jogo, é necessário instalar os seguintes
pacotes (SFML, Cmake, Make, Threads) com o comando abaixo no terminal:

sudo apt-get install -y build-essential cmake libsfml-dev libpthread-stubs0-dev

Depois de instalados os pacotes, basta executar
o comando abaixo dentro da pasta que se encontra o
arquivo "run.sh" para compilar o jogo:

bash run.sh
